# Readable Tracing Art Mirror

This directory is generated by `npm run deoptimize`.

It is a full parallel copy of `../site/`:

- JS, CSS, JSON, and web manifest assets are formatted into readable files.
- SSR HTML, SVG, XML, and binary assets are copied byte-for-byte because
  whitespace can affect hydration-sensitive Nuxt output.
- Binary assets are copied byte-for-byte.
- The original scraped runtime in `../site/` is not modified.

The goal is to make the Nuxt bundle easier to inspect and gradually turn into a
maintainable source of truth later. The generated files are still compiled
runtime code, not recovered Vue source components.

## Serve The Readable Copy

```bash
SITE_DIR=readable/site PORT=5175 npm run serve
```

Then verify with:

```bash
TARGET_URL=http://localhost:5175/tracingart/ npm run verify:puppeteer
TARGET_URL=http://localhost:5175/tracingart/ npm run verify:mobile
```
